Learning Web Development: HTML, CSS & JavaScript

A beginner-friendly guide to building your first interactive webpage

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating webpages. It provides the structure and content of your site using different elements.

  • <h1> - Main heading of the page.
  • <p> - Paragraphs for text content.
  • <img> - Displays images. src is the image link, alt is alternative text for accessibility.
  • <a> - Creates links to other pages or sections.

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout your webpage. It controls colors, fonts, spacing, and even responsive design for different devices.

  • background-color - Sets the background color of the page.
  • font-family - Changes the font style.
  • margin and padding - Add space around and inside elements.
  • @media - Makes your site responsive for mobile devices.

Notice how this page looks good on both desktop and mobile? That's thanks to CSS!

What is JavaScript?

JavaScript is a programming language that lets you add interactivity to your webpage. You can respond to user actions, update content, and much more.

  • <button> - Adds a clickable button.
  • onclick - An event that runs code when the button is clicked.
  • alert() - Shows a popup message.
